# intelliMotors

Intrucciones para correr el programa

Iniciar Servidor
  -Abrir carpeta server
    $cd server
  
  -Instalar los modulos de node
    $npm install
  
  -Iniciar servidor
    $npm run dev
  
 Iniciar Cliente
  -Abrir carpeta cliente
    $cd cliente
  
  -Instalar los modulos de node
    $npm install
  
  -Iniciar cliente
    $npm run dev
